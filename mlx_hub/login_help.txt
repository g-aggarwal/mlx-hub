Missing: Hugging Face - User Access Token

MLX-Hub uses `huggingface_hub` to interact with Hugging Face MLX models.
Please add an access token from Hugging Face to huggingface_hub to proceed.

Hub documentation:
> https://huggingface.co/docs/hub/security-tokens

To create an access token, go to Hugging Face settings:
> https://huggingface.co/settings/tokens

To add the access token to `huggingface_hub`:
> huggingface-cli login
